# Gallery Images

Place your gallery images in this directory.

## Image Requirements

- Format: JPG, PNG, or WebP
- Recommended size: 1200x1200px or larger
- Naming convention: `image-1.jpg`, `image-2.jpg`, etc.

## Adding Images

1. Add your images to this directory
2. Name them sequentially: `image-1.jpg`, `image-2.jpg`, etc.
3. The Gallery component will automatically load them

## Placeholder Images

If you need placeholder images for development, you can use:
- https://picsum.photos/1200/1200 (random images)
- Or create simple colored placeholder images
